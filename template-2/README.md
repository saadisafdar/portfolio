# Template 2 — Saadi Portfolio

A full-stack developer portfolio landing page built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Sections

1. **Hero** — Massive gradient heading with magnetic portrait effect
2. **Marquee** — Scroll-driven dual-row image marquee
3. **About** — Character-by-character scroll-reveal text with decorative 3D corner images
4. **Services** — White section listing 5 numbered services
5. **Projects** — Sticky-stacking project cards with scroll-driven scaling
