# Saadi Safdar - Portfolio Website

This is my personal portfolio website built with Gatsby.js, showcasing my work as a Computer Science student, full-stack developer, and AI enthusiast.

## Features

- **Modern Design**: Clean, responsive design with smooth animations
- **Project Showcase**: Display your featured projects with images and descriptions
- **Experience Section**: Highlight your work experience and skills
- **Contact Form**: Easy way for visitors to get in touch
- **Social Media Integration**: Links to all your social profiles

## Technologies Used

- [Gatsby.js](https://www.gatsbyjs.com/) - Static site generator
- [React](https://reactjs.org/) - Frontend framework
- [Styled Components](https://styled-components.com/) - CSS-in-JS
- [GraphQL](https://graphql.org/) - Data querying

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/saadisafdar/portfolio.git
   cd portfolio/template-2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run develop
   ```

4. Open your browser to `http://localhost:8000`

### Build for Production

```bash
npm run build
npm run serve
```

## Project Structure

- `src/` - Main source code
  - `components/` - React components
  - `config.js` - Site configuration (email, social media, etc.)
  - `styles/` - Global styles and themes
- `content/` - Markdown files for projects and experience
  - `jobs/` - Work experience
  - `projects/` - Regular projects
  - `featured/` - Featured projects with images
- `static/` - Static assets (images, PDFs, etc.)

## Customization

To customize this portfolio for your own use:

1. Update `src/config.js` with your personal information
2. Add your projects to the `content/projects/` directory
3. Add featured projects to `content/featured/` with images
4. Update your work experience in `content/jobs/`
5. Replace the profile image in `src/images/me.jpg`
6. Update the favicon and other images in `src/images/`
