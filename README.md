## ✨ About

This repository serves as the central hub for my portfolio development. Rather than maintaining a single design, I continuously experiment with new layouts, technologies, animations, and user experiences.

Each portfolio version represents a different stage of my learning and development journey, allowing me to explore new ideas while improving existing ones.

As I learn new technologies and design patterns, additional portfolio versions and features will be added to this repository.
